


# trl command 

Transformer Reinforcement Learning

```

trl sft: fine-tune a LLM on a text/instruction dataset
trl dpo: fine-tune a LLM with DPO on a preference dataset
trl chat: quickly spin up a LLM fine-tuned for chatting

```
