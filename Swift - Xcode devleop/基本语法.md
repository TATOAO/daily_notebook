
# 基本语法

## 变量定义

```
var y:String = "hello"
let x:Int = 3

var s:Character = "a"
```

变量声明之后不能再定义


单引号不能表示 string

## switch 语法

``` swift

switch x {
case "1" :
    // code 1
case "2", "5" :
    // code 2
default :
    // code 3
}

```
