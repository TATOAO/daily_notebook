
# 在没有网络的环境安装nvim



## linux 


### 安装 nvim
首先直接安装nvim本体, 

[github neovim release](https://github.com/neovim/neovim/releases/latest ":)")

nvim-linux64.deb 

然后在环境里 apt install ./nvim-linux64.deb


### 安装其他包

 open ~/.local/share/
 把里面的nvim文件夹直接打包压缩，然后到目标机器的同一个目录里解压
