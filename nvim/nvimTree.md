
## 快捷键


### cd 

```
-                   # 上一级 cd ..
<C-[>               # 下一级 cd , control + [

```

### display
```
W                   #  大写 W Collapse_ALL
E                   #  大写 E Expand_ALL


tab                 # preview
```


### Execution

```

.                   # leave run command cursor at front

```


### edit


```

c                   # copy the name only
y                   # copy the full path

o                   # open and edit

s                   # system open

```


### search 

```
S
```
