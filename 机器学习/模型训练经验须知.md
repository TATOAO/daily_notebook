# 训练模型

最近遇到的问题是，在jupyter上面训练模型，到最后每次调整的参数，保存的模型，非常乱，代码到最后不知道哪里改了哪里，怎么比较，新的对比指标可能前期又没有固定。

所以需要一个范式，或者说经验来规范之后的代码和效率。

(model - dataset) 的不同组合有一个 - evaluation

model 和 dataset 不是通用的，model有规定的输入数据模式 

``` py
class model
    - model id
    - model meta 
    - train time
    - train data
    - save locationß
    - evaluations
    
class dataset
    - dataset name
    - save location
    - build time
    - evaluations
    - build methods

# not to over use data, 

class evaluation
    - model name
    - dataset name
    - detail evaluations
```
