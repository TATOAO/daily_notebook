# 基础协议

## SPP Server and Client


**SPP** Serial Port Profile（串口配置文件）

其定义了使用蓝牙进行RS232（或类似）串行电缆仿真的设备应使用的协议和过程。简单来说就是在蓝牙设备之间建立虚拟的串口进行数据通信。
